# R200 Performance, Latency, and Simplification Audit

Primary source was static repo inspection plus the checked-in bundle baseline in
`scripts/bundle-baseline.json`. A fresh Vite build was attempted with
`npx vite build --configLoader runner`, but the sandbox blocks Vite's temp write
to the parent repo `node_modules/.vite-temp`; the attempt still surfaced useful
chunking warnings noted below.
