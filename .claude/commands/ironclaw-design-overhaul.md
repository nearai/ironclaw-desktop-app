# IronClaw Design Overhaul

Run the prompt in `docs/prompts/ironclaw-design-overhaul.md`.

This is the full design overhaul workflow:

1. Read the design overhaul notes and current screenshots.
2. Produce the required design documents first.
3. Select one highest-value implementation slice.
4. Patch only that slice.
5. Refresh screenshots and run the required static verification.

Keep the product thesis intact: IronClaw Desktop is a prepared-desk chief of
staff, not a generic chat client.

Do not reintroduce generic provider setup, fake connector readiness, huge gray
placeholder text, warm SaaS styling, or workflow sludge in chat.
