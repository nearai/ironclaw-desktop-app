# IronClaw Design Pattern Generator

Run the prompt in `docs/prompts/ironclaw-design-pattern-generator.md`.

Generate product patterns for IronClaw Desktop as an agentic chief of staff.
Do not produce generic chat-app advice. Every pattern must include:

- Surface
- User job
- Visual structure
- Interaction behavior
- Data contract
- Honest blocked state
- Test or screenshot proof

End by naming the first three patterns to implement.
